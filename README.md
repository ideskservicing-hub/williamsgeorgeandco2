# Williams George & Co. (Chartered Accountants)

Strategic Financial Excellence - Chartered Accountants providing comprehensive financial solutions.

## Repository
[https://github.com/ideskservicing-hub/williamsgeorgeandco2.git](https://github.com/ideskservicing-hub/williamsgeorgeandco2.git)

## Tech Stack
- React 19
- Vite
- TypeScript
- Tailwind CSS
- Motion (Framer Motion)
- Lucide React

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/ideskservicing-hub/williamsgeorgeandco2.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
